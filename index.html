Portfólio

Projeto Stock Code

Stock Code é uma API desenvolvida em Java com Spring Boot, utilizando MySQL para persistência de dados e Docker para ambiente de execução.

A aplicação foi criada com o objetivo de facilitar o gerenciamento de estoque para pequenos empreendimentos.

  • Cadastro de marcas, categorias e produtos
  • Controle de entrada e saída de produtos
  • Registro de transações de compra e devolução

O projeto foi construído do zero, desde a modelagem do banco até a estrutura da API, com foco em organização, clareza de código e aplicação de boas práticas.

Projeto Portfólio Pessoal

Desenvolvi este projeto com o objetivo de apresentar meus trabalhos e consolidar meus conhecimentos em front-end moderno.

O layout foi criado no Figma e implementado utilizando React, TypeScript e Next.js. A interface é composta por cards com efeito de vidro fosco (glassmorphism), criando uma aparência moderna e limpa.

  • React
  • TypeScript
  • Next.js
  • Styled Components
  • Framer Motion (animações de transição)
  • Lucide (ícones em SVG)

O projeto foi importante para consolidar conceitos de componentização, animações e organização de estrutura em aplicações React.
